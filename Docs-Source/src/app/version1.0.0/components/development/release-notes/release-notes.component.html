Since this is the first release of the DotNet Validator library, we will include all features that has been developed with our lib. here.

Features

  • A punch of validation utilities to validate .NET data-models.
  • A punch of sanitization utilities to re-format .NET data-models values before saving it.
  • DotNet Validator lib. supports .NET Standard 2.0 to be available a cross a wide range of Microsoft platforms.
