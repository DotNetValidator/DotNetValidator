A month after that i was working on a .NET backend and i used to do all data-model validations line-by-line property-after-property by my self, and i stopped for a second.
“Why we don't have a .NET library like this great one?!”.
So, i decided to start work one a .NET library that makes data-model validation simple & easy to use.
