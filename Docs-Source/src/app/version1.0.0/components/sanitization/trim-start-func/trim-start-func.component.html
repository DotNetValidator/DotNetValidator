An extension sanitization method that removes all leading occurrences of a set of characters from the current validated property.

Definition

  • .TrimStart(params char[] trimChars)

    Supported data types :

    • Strings.

    Parameters :

    • trimChars : An optional params set that will be removed from the leading of the string.
      If omitted, all leading white-spaces will be removed.

    Returns :

    • A Validator to add more validation or sanitization.
