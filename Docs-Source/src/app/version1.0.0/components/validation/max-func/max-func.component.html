An extension validation method checks if the property's value is not more than the given maximum value.

Definitions

  1. .Max(double maxValue, string errorMessage = null)

    Supported data types :

    • Doubles.
    • Decimals.
    • Floats.
    • Singles.
    • Longs.
    • Integers.
    • Shorts.
    • Bytes.

    Parameters :

    • maxValue : A numeric maximum value.
    • errorMessage : An optional validation error message.
      If omitted, a default error message will be used.

    Returns :

    • A Validator to add more validation or sanitization.

  2. .Max(DateTime maxDateTime, string errorMessage = null)

    Supported data types :

    • DateTimes.

    Parameters :

    • maxDateTime : A maximum date-time value.
    • errorMessage : An optional validation error message.
      If omitted, a default error message will be used.

    Returns :

    • A Validator to add more validation or sanitization.

  3. .Max(TimeSpan maxTime, string errorMessage = null)

    Supported data types :

    • TimeSpans.

    Parameters :

    • maxTime : A maximum time-span value.
    • errorMessage : An optional validation error message.
      If omitted, a default error message will be used.

    Returns :

    • A Validator to add more validation or sanitization.
