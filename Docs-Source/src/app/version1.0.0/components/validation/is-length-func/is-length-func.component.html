An extension validation method that validates the length of the property's value using the given minimum and maximum length.

Definition

  • .IsLength(long minLength, long maxLength, string errorMessage = null)

    Supported data types :

    • Strings.

    Parameters :

    • minLength : The minimum length.
    • maxLength : The maximum length.
    • errorMessage : An optional validation error message.
      If omitted, a default error message will be used.

    Returns :

    • A Validator to add more validation or sanitization.
