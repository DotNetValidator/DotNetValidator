An extension validation method checks if the property's value is not less than the given minimum value.

Definitions

  1. .Min(double minValue, string errorMessage = null)

    Supported data types :

    • Doubles.
    • Decimals.
    • Floats.
    • Singles.
    • Longs.
    • Integers.
    • Shorts.
    • Bytes.

    Parameters :

    • minValue : A numeric minimum value.
    • errorMessage : An optional validation error message.
      If omitted, a default error message will be used.

    Returns :

    • A Validator to add more validation or sanitization.

  2. .Min(DateTime minDateTime, string errorMessage = null)

    Supported data types :

    • DateTimes.

    Parameters :

    • minDateTime : A minimum date-time value.
    • errorMessage : An optional validation error message.
      If omitted, a default error message will be used.

    Returns :

    • A Validator to add more validation or sanitization.

  3. .Min(TimeSpan minTime, string errorMessage = null)

    Supported data types :

    • TimeSpans.

    Parameters :

    • minTime : A minimum time-span value.
    • errorMessage : An optional validation error message.
      If omitted, a default error message will be used.

    Returns :

    • A Validator to add more validation or sanitization.
