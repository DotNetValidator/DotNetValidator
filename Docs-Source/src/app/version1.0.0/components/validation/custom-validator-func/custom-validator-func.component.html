An extension validation method that creates a custom validation method that accepts an object or custom-type value and returns true if there is no validation error, otherwise false.

Note : Any mutation to the value will not affect the original value.

Two Definitions

  1. .CustomValidator(Func<object, bool> customValidator, string: errorMessage)

    Supported data types :

    • Any data type.

    Parameters :

    • customValidator : A custom validator that accepts an object value and returns true if there is no validation error, otherwise false.
    • errorMessage : A validation error message.

    Returns :

    • A Validator to add more validation or sanitization.

  2. .CustomValidator<TSource>(Func<TSource, bool> customValidator, string: errorMessage)

    Supported data types :

    • Any data type.

    Parameters :

    • customValidator : A custom validator that accepts an TSource value and returns true if there is no validation error, otherwise false.
    • errorMessage : A validation error message.

    Returns :

    • A Validator to add more validation or sanitization.
