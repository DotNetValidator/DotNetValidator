An extension validation method checks if the property's value contains the given string value, or at least one value from the given array of values.

Two Definitions

  1. .Contains(string str, string errorMessage = null)

    Supported data types :

    • Strings.

    Parameters :

    • str : A string value that the property's value should contains.
    • errorMessage : An optional validation error message.
      If omitted, a default error message will be used.

    Returns :

    • A Validator to add more validation or sanitization.

  2. .Contains(string errorMessage = null, TParam[] strs)

    Supported data types :

    • All primitive data types.

    Parameters :

    • errorMessage : An optional validation error message.
      If omitted, a default error message will be used.
    • strs :The values that the property's value should contains one at least.

    Returns :

    • A Validator to add more validation or sanitization.
