A public static class ValidationResult that resolves the DotNet Validator errors.

Members

  • Validate(params Validator[] validators)

    A public static List<ValidationError> method that validates the given set of Validators and returns a List<ValidationError> if there is at least one validation role failed or returns null if all validation roles passed.

    Parameters :

    • validators : params Validator[] of validators to be validated.

    Returns :

    • A List<ValidationError> if there is at least one validation role didn't passed.
    • Or null if all validation roles passed.

  • Result(params Validator[] validators)

    A public static dynamic object method that validates the given set of Validators and returns a dynamic object if there is at least one validation role failed or returns null if all validation roles passed.

    Parameters :

    • validators : params Validator[] of validators to be validated.

    Returns :

    • A dynamic object if there is at least one validation role didn't passed.
      This dynamic object will contain each non-passed property and each property value will be the list of validation error messages as a List<string>.
    • Or null if all validation roles passed.
