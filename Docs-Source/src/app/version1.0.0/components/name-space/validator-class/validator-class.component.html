A public class Validator that holds all of validation errors for a given Property under a specific data model against all validation roles associated with it.

Validator class is the most important part in the DotNet Validator lib.

Although this class is not a static class, you can't create a new instance of it by your self, because it's constructor is marked as internal.

Members

  • Create(object data, string propertyName)

    A public static Validator method that creates a new Validator that can be used to validate or sanitize the given Property value using some validation or sanitization utility methods under the given data model.

    Parameters :

    • data : The data model of type object.
    • propertyName : The property name to be validated.

    Returns :

    • A new instance of Validator for the given Property under the given data model.
