The ValidationResult contains to different static methods those are used to validate data-models, the first one is ValidationResult.Validate() and the other one is ValidationResult.Result()

How to Use?

  • Using ValidationResult.Validate() :

    To do :
    1. Validate that the person's name length is not more than 5 characters.
    2. Validate that the person's age must be at least 20 years.

    Let's start with creating a new class called Person. 

                            {{person1}}
                    
                            {{validate}}
    • The result will be a List<ValidationError>.
    • Each item in the list contains two properties :
      1. PropertyName : the name of the property in the data-model.
      2. ErrorMessages : the list of error messages resulting from validating the property.

  • Using ValidationResult.Result() : 

                            {{result}}
    • The result will be a dynamic object.
    • Each property that had a validation error will be added to this dynamic object.
    • the value of the property is the list of error messages resulting from validating the property.
