An extension sanitization method that creates a custom sanitizer method that accepts a generic-type value and returns the same generic-type value that has been modified using the custom sanitizer.

Definition

  • .CustomSanitizer<TSource>(Func<TSource, TSource> customSanitizer)

    Supported data types :

    • All data types.

    Parameters :

    • customSanitizer : A custom sanitizer that accepts an TSource value and returns a new TSource value that has been modified.

    Returns :

    • A Validator to add more validation or sanitization.
