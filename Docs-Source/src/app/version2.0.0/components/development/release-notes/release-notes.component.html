This version has no new features, but has some breaking changes.

Features

  • No new features added.

Breaking Changes

  • Change .IsLatLang() validation method to .IsLatLong().
  • Remove the .ToDateTime() sanitization method because it's not useful to use in .NET Programming Languages .
  • Remove the unnecessary overloads from the .CustomValidator() validation method & the .CustomSanitizer() sanitization method, since both are accepting a generic-type and there is no need for any overloads that takes a static-type.
  • Change str parameter word to token in .Contains(), .StartsWith() and EndsWith() validation methods.
