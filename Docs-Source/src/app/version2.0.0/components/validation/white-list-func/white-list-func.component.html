An extension validation method that checks if the property's value is in the given white-list.

Definition

  • .WhiteList(TParam[] whiteList, string errorMessage = null)

    Supported data types :

    • All primitive data types.

    Parameters :

    • whiteList : An array of white-list values of any primitive data type.
    • errorMessage : An optional validation error message.
      If omitted, a default error message will be used.

    Returns :

    • A Validator to add more validation or sanitization.
