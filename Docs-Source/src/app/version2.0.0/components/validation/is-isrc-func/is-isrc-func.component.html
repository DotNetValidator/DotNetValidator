An extension validation method checks if the property's value is a valid international standard recording code .

Definition

  • .IsISRC(string errorMessage = null)

    Supported data types :

    • Strings.

    Parameters :

    • errorMessage : An optional validation error message.
      If omitted, a default error message will be used.

    Returns :

    • A Validator to add more validation or sanitization.
