An extension validation method checks if the property's value is a valid ip address.

Definitions

  1. .IsIP(string errorMessage = null)

    Supported data types :

    • Strings.

    Parameters :

    • errorMessage : An optional validation error message.
      If omitted, a default error message will be used.

    Returns :

    • A Validator to add more validation or sanitization.

  2. .IsIP(IPVersion ipVersion, string errorMessage = null)

    Supported data types :

    • Strings.

    Parameters :

    • ipVersion : The version of the ip address.
    • errorMessage : An optional validation error message.
      If omitted, a default error message will be used.

    Returns :

    • A Validator to add more validation or sanitization.
